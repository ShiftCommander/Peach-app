PEACH GUITAR TUNER V33 — INSTALLATION ANDROID

1. Dézippe le dossier peach-guitar-tuner-v22.
2. Dépose tout le contenu du dossier sur Netlify, à la racine du site.
3. Ouvre l’URL HTTPS Netlify dans Chrome sur Android.
4. Autorise le micro.
5. Installe l’app depuis le bouton proposé dans Peach ou via Chrome ⋮ > Installer l’application / Ajouter à l’écran d’accueil.

IMPORTANT SI TU REMPLACES UNE ANCIENNE VERSION
- Désinstalle l’ancienne PWA Peach de ton téléphone ou vide le cache du site dans Chrome.
- Recharge l’URL Netlify après déploiement.
- Le service worker est passé en peach-guitar-tuner-v22 pour forcer la mise à jour du cache.
